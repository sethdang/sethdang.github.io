    2004-12-23

    2004-12-23 阴 星期四 呆在家中，停电了很久,想到很久没写日记了，也没去买个好的日记本，在个破本上就开始了。 最近很多事想记下来，更想忘记，矛盾！ 决定离开重庆，又觉得舍不得（比我离开成都还舍不得），临到真的打算离开又犹豫什么呢？自己变得优柔了吗！最近老在自我否定，怀疑自己是不是定错了目标？是！怀疑自己是不是入错了行？是！怀疑自己是不是选错了地方？是！自己还是以前的自己？不是！同样的问题问过自己n遍之后才发现自己已经变了，尽管还有一摩尔的物质是不变的。 不断的否定中，我丧失了自信、以及很多优良的品质。我不再那么天真无邪，不再有什么信仰，不再有什么追求（或许不久也会有）。还是坚信一句话“时间时疗伤的良药”时间能帮我解决很多问题，我能依靠的也仅仅是时间了。 重庆真的留下了我很多思绪，快乐，有最快乐的时光，痛苦，甚比撕心裂肺；事业、成就也都是昙花一现。凡是给我带来过这么多回忆的城市我是没有理由恨她的，尽管我并不喜欢重庆，但我一辈子也忘不掉了。她呢？我这个人不是很爱憎分明，本来想真真正正的爱一次&恨一次！结果爱过了，恨过了。 留下的只是回忆和淡淡的痛。也无所谓了。 16：19电还在继续停，光线不怎么好了。OVER！ 2004-12-26抄

    [Daily life]雨一直下

    天气:雨 心情:差 2006－3－24 雨一直下、下好大。老大休假了，我確並不輕松！ 早上剛好在雨沒怎麼下的時候騎了車過來，運氣還不算太壞。看了昨天貝貝給我的信，我好好心痛。愛她都迷失了自己，我不敢也不要願意多想。繼續我辛勤的工作！ bbs上發出的通告讓我對工廠好絕望，有我們老大瞎搞，我們都沒自由和利益了。心裡面好恨。郁悶吧，在一個雨天。

    [Daily life]吵架、生氣－＞投降了

    天气:雨 心情:差 2006－3－24 和同事阿松吵架了,自己的事情自己作！作不好自己挨罵去，看不起那號人也不和他計較。 下班了，貝貝問起這件事情，隨便說了兩句。不想說得太多，怕大家傷了感情、怕她為我操心，我知道貝貝很關心我。問了幾句，居然問我是不是被欺負了…自尊心嚴重受辱、有點吃醋，干嗎老認為我不如別人呢。無名業火冒了出來，我凶了一句。貝貝臉色一下變了 去吃飯，什麼也沒吃，看她哪樣冷落我我心裡面象刀割一樣。我哪裡吃得下，點了二十八圓的東西一點沒吃。想了很多，知道自己錯了：不該凶自己的愛人，一直放下尊嚴求她原諒，我這輩子都沒有那麼去求過人。她並沒有什麼表現，我心真的好痛… 我想多說無益，是我的錯我投降，如果真的那麼不理解我也認了。甚至我想了好多。沒有再管，騎車上課去了，一路上車和人都很少，還好沒有出交通事故。我不想去上課了，可我知道不去上課自己會更難過，到學校了同學都很活躍，說那麼多外語我都沒聽懂。腦袋裡面一片空白，唯一想的就是早些下課好去接我的貝貝，想到她那麼冷漠就心好痛。甚至懷疑她是不是有我想的那麼愛我。不管那麼多了，我只知道我太愛她了愛到沒有了自我。我投降…不管她有什麼反應。
