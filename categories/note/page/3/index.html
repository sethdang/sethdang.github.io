千岛湖第一日

为了躲避台风、我的行程从南麂岛改到了千岛湖。尽管这样，心里还是担心台风的到来会影响行程！还好至少到现在为止还没有受到天气的侵扰。千岛湖是两江一湖（富春江、新安江、千岛湖）国家级重点风景名胜区的主要组成部分。它地处长江三角洲的腹地，是上海和杭州的“后花园”。 起了个大早，7：30吃完早餐就在温州踏上了旅游大巴开始了我的千岛湖之旅。我们的第一个目的地是位于建德市石屏乡境内的灵栖洞，建德也是我们这次出行的居所。大巴辗转了数条高速，10：54终于在航新景高速的航头收费点驶出。汽车开在田间小道上，路面是单车道最多只容得下两辆汽车，经过半个多小时的颠簸，我们的第一个目的地灵栖洞到达了。 灵栖洞由灵泉、 清风、霭云三洞和灵栖石林组成！正午了，第一个目标便成吃饭，怀着企盼的心情在一顿狼吞虎咽后开始了我们的旅程，景区只有唯一的一个饭店，口味并不怎么样。听导游说霭云洞由于正在维修，暂时没有开放，灵泉洞则是一个需要座船进入的溶洞。参观心切，跑在前面的我们被景区的人误导到了一座小寺庙，挨个儿发了香烛，当然香烛不是白烧的，还要捐点功德。一个MM身上没有零钱，没有办法：为了不丢面子，只有丢大钞罗。我还好，吃早餐还剩了些钱，刚好抵用，黑呀！ 灵泉洞口，导游召集齐人马13人一船进洞去。洞口的潭水不够那么赏心悦目，一路上的流水却很清澈和冰凉，洞里景色还是挺不错，石钟乳在灯光的映衬下显得也特别的壮观，石阶挺多，到处落差很大，有几个厅是传说中西游记的拍摄地点。唯一的缺陷就是所有的石钟乳因为长期的开放已经泛黑了，我有见过纯白的石钟乳，所以对这样黑黑的东西有点不感冒。 清风洞口真是凉风阵阵，入口前还有一个隔厅，由于温差产生的自然风吹起来也是特别的爽。洞内温度比较低，在隔厅适应很久，进去才勉强适应了。 石林到了，进入其中简直迷宫一样，呵呵入口处就是写的迷宫哦。实在走了会儿，我们选择了在石头上面攀爬了，挺有味道的。 13：45参观完灵栖洞后我们团再次集合到了一起，开往我们的千岛湖上幸运岛！路上路过建德市区，停车到一个超市购物，跟团就这点弊端。建德在我看来是一个挺干净的小城，人口密度和车流密度都比较少。超市卖的是倒笃菜和一些果脯类的干货，当然我是没有买，不会做，会做也不一定爱吃! 大约又过了半小时，我们的旅行车开过一个漆黑的单行隧道，到达了码头。从码头乘坐游轮到好运岛，在好运岛我们登到了最高峰，俯瞰了岛上的风景，很普通的一个岛屿，没有什么特色。在岛上逗留了一个半小时左右我们有些失望的回到游轮，才知道原来游轮上的晚餐和新安江水库的游览才是此行的重点。 游轮行驶在宽阔的千岛湖上，我站到船头甲板上，那习习凉风带着润湿的水气和一些上空聚集的雨点轻柔的抚摸着我，皮肤上一种润湿的黏黏的感觉。两岸无数的小岛才让我明白，刚才游览那个幸运岛原来也是身在其中的无数个小岛中典型的一个。听说很多小岛都是无人岛，没有被开发的。当然千岛湖是一个人工湖，因为兴安江水库的修筑才形成了这1千多个小岛。我们的游轮正是要开往兴安江水库大坝。 途中我们开始用餐了，千岛湖的鱼头比较出名，不多游轮上的比较贵和比较少，我们几个驴友商定一会儿回宾馆后出来吃夜宵！晚餐还行，也有月饼（小号的）和水果。不过还是期望夜宵再填饱肚子。 兴安江水库终于在餐后，拍完照以后到达了，停留些许，便返程了。 18：44我们再次踏上了我们的返程大巴，开赴传说中的三星宾馆。大家都在讨论是不是也个名叫三星的宾馆？让我们失望了，的确不是一个叫三星的宾馆，虽然设施还可以，终究没有我想象的那么好，还好有电脑用也，不过时间好紧，明早7：00要集合，晚上还要去夜宵呢。 打了车到市区政法路去吃鱼头，喝夜啤，我破天荒的喝了一瓶多酒，也抽了几支烟。哇塞，多么充实的一天！

玫瑰

“送人玫瑰，手有余香。” 公司同事中秋晚会聚餐后，陪同事一起去买了一束玫瑰。 想起了。。。三年前我买的那朵玫瑰，那朵已凋谢很久的玫瑰。 这么几年了，该放下的东西还没有放得下，该拾起的也还没有拾起。 想起了。。。她发来的短信，我没有回复。 我有太多的话，千言万语，我选择用沉默表达，她能幸福是我最大的心愿。 想起了。。。我的家人，他们的电话我也没有应答。 我很内疚，可我不知道该说些啥，我对自己已经无言应答。 想起了。。。我的弟弟，所有的祝福我都不善表达。 他也开始了他人生的脚步，兄弟，走好。一切都要靠自己去闯。 想到了。。。我明天的远足 我明天就要远足，出去玩一下，其实我已经走得很远，我也很想家。

赌局

最近股市惨淡，我却神奇的一连卖了两次当日最高价，算是我这辈子少有的奇遇了。又套进去了不少，也在一来一去中享受这现实的折磨同煎熬。说来我这个人倒是真没有睹运，从小到大很少赢过什么赌局，不过赌性倒是哪个人都有，我也算控制得很好的,赌博的事我很少参与。 在人生很郁闷的时候我选择了炒股，希望一点点的刺激能还给我对生活的激情,可是世事不如意十之八九，我站在5320点的高位买入第一支沪市的股票，然后等待我的是股市一路暴跌。如今上证指数站到了2770，多次倒手我从一个菜鸟变得成熟，股市的风云变幻让我更清醒的认识到现实的残酷。跌得很惨烈，我也学会了认赔出局的勇气。 人生呢？我仿佛是一个不会认赔出局的赌徒。被自己套在一支票上，全仓买入.后来还发现亏损了，ST了，破产清算了。我的一切还能拿得回来吗？

钱夹

一年多时间我换了三个钱夹罗,钱夹质量怎么会那么差?再换钱夹的时候,看着钱包仅有的几个钱,只能买个普通的先用着,赚了钱再换吧! 跑了好多地方,大学毕业以前还没有用钱夹的习惯,毕业后好友送了一个华伦天奴的钱夹给我,开始有了点小钱正好用上了.然后一路的漂泊让我的钱夹一会儿满一会儿空.世事浮沉,我在蹉跎流失的岁月中成长,强壮,恋爱,失恋,失意……每一步都洗礼着我的钱夹,让它也留下了岁月的痕迹.终于,在和珍珍道别的时候我的华伦天奴退役了.开始了我频繁的更换钱夹的历程. 前些天去嘉兴,我用的登喜路的钱夹(应该是冒牌的哟)坏掉了,郁闷呀,又要花钱去买. 怀恋我过去用过的钱夹,新买的钱夹我要用他去装更多的钱.
